## About The Project

<a href="https://writeai.app">
    <img src="assets/images/title.png" alt="Logo" height="50">
  </a>

Write AI is a text editor developed with Flutter for iOS and Android.
Powered by OpenAI's GPT-3, Write AI uses artificial intelligence to predict your next sentence.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [Flutter](https://flutter.dev/)
* [Firebase](https://firebase.google.com/)
* [OpenAI][https://openai.com/]
